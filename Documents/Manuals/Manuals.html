IBM 1620 Manuals

This section of the website contains IBM's manuals for the IBM 1620. 129 different manuals are known, many of which had multiple versions and/or updates released.

Every IBM 1620 manual had a publication number of the form: xxxx-yyyy-z
    where: xxxx - designated the type of manual and which group within IBM released it
                yyyy - was a unique 4-digit document number
                      z - was the revision number of the document

It was common in the early days of the IBM 1620 that the first release of a manual would omit the "-z" revision number, assuming it to be "-0". Later, first revisions explicitly used "-0".

Occassionally, different groups would release the same manual, but all of them used the same 4-digit document number.

Documents describing software were treated differently and are here.
