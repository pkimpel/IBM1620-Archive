    <div id=BodyText>

        <p>This section of the website contains IBM's manuals for the IBM 1620.  129 different manuals are known, many of which had multiple versions and/or updates released.</p>

        <p>Every IBM 1620 manual had a publication number of the form: <b><i>xxxx-yyyy-z</i></b><br>
           &nbsp; &nbsp; where: <b><i>xxxx</i></b> - designated the type of manual and which group within IBM released it<br>
                 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; <b><i>yyyy</i></b> - was a unique 4-digit document number<br>
                 &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; <b><i>z</i></b> - was the revision number of the document</p>

        <p>It was common in the early days of the IBM 1620 that the first release of a manual would omit the "-z" revision number, assuming it to be "-0".  Later, first revisions explicitly used "-0".</p>

        <p>Occassionally, different groups would release the same manual, but all of them used the same 4-digit document number.</p>

        <p>Documents describing software were treated differently and are <a href="">here</a>.</p>
